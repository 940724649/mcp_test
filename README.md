# MCP 测试项目

这是一个使用 Model Context Protocol (MCP) 创建的测试仓库。

## 项目介绍

MCP 是一个允许 AI 模型与各种服务进行交互的协议，包括：
- 文件系统访问
- GitHub 仓库操作
- 等其他功能

## 目录结构

```
mcp_test/
├── README.md         # 项目说明文档
├── src/              # 源代码目录
│   └── main.py       # 主程序
├── tests/            # 测试目录
│   └── test_main.py  # 测试文件
└── requirements.txt  # 项目依赖
```

## 如何使用

1. 克隆仓库
2. 安装依赖
3. 运行程序
